Notion ‚Üî Asana Task Sync

Input

Trigger from Notion Webhook when a task is created or updated.

Tools Used

  • n8n (Logic)
  • Notion API
  • Asana API
  • Slack API

Workflow Logic

When a task updates in Notion, the system normalizes the data. Switch nodes route the task based on status (e.g., "In Progress" moves to Asana "In Progress"). If IDs are missing, it triggers a Slack alert. Completed tasks send a summary notification.

"I wanted something that felt invisible‚Äîwhere updating a task in Notion would automatically nudge Asana into the right state, without anyone having to copy-paste. The goal was peace of mind."

Short-Form Video Generator

Purpose

Automates creation of marketing videos by transforming TikTok transcripts into AI-narrated videos.

Tools Used

  • Apify (TikTok Scraper)
  • GPT-5 (Scriptwriting)
  • HeyGen (Video Gen)
  • Google Drive

Workflow Steps

1. Scrape TikTok transcript.
2. Rewrite script using GPT-5 (Storytelling mode).
3. Send to HeyGen to generate Avatar video.
4. Upload to Drive and log in Sheets.

"I wanted to make sure it could run end-to-end without human intervention. I focused on error handling and added a cap of 15 posts to keep API costs predictable."

Lead Enrichment System

Input

Webhook with Company URL and Name.

Tools Used

  • Apollo API
  • Serper (Google Search)
  • Gemini (Analysis)
  • Apify (LinkedIn)

Logic

The system queries Apollo for financial data, scrapes the company website for context, and uses Serper/Gemini to find and summarize the LinkedIn profile. It compiles everything into a styled Google Doc.

"Data lives in so many places. I wanted a single assistant that could gather essentials, check accuracy, and hand back a polished doc that sales can use immediately."

Voice Agent Lead Qual

Input

New Lead from Tally Form.

Tools Used

  • Vapi.ai (Voice)
  • Twilio
  • Cal.com
  • GPT-4o

Workflow

Webook triggers an outbound call via Vapi. The AI qualifies the lead. If qualified, it checks Cal.com for slots and books the meeting live on the call. Follow-up emails are sent automatically based on outcome.

"My objective was a friction-free pipeline: form fill ‚Üí booked, qualified meeting in under three minutes."

Weekly Marketing Reports

Input

Schedule Trigger (Monday 7am).

Tools Used

  • Google Analytics 4
  • Google Ads API
  • Meta Ads API
  • GPT-4o (HTML Formatting)

Logic

Fetches data from 5 different domains + Ads platforms. Normalizes metrics. Uses GPT-4o to write a summary analysis and format it into a clean HTML email table.

"Pulling data manually was tedious. I wanted consistency and clarity, meeting the team where they work: detailed insights via email, quick updates in Slack."

Hotel Outreach System

Input

Manual Trigger + TripAdvisor Query.

Tools Used

  • Apify (TripAdvisor)
  • mails.so (Verification)
  • GPT-5 (Copywriting)
  • Gmail

Workflow

Scrapes 750+ hotels. Cleans data. Validates emails. GPT-5 writes a personalized 90-word pitch. Creates a Draft in Gmail for human review.

"By validating addresses first, token spend is minimized and bounce-risk drops. Draft-only sending keeps a human in control while scaling volume."
